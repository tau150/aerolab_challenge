## Aerolab challenge.


### Products catalog  

App made for Aerolab challenge where you can see and redeem some catalog's products.

### Libraries used for this project:

 - React.
 - Redux.
 - Redux-Promise.
 - Redux Thunk.
 - Styled Components.
